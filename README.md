# README

## Installation

To use this software, follow these steps:

1. Clone the repository to your local machine.
2. Ensure you have a C++ compiler.
3. Compile the source files using your preferred C++ compiler.
4. Run the compiled executable.

## Usage

The software provides the following functionality:

1. **Date Management**: The `clsDate` class allows you to create, manipulate, and perform various operations on dates, such as:
   - Constructing dates from different input formats
   - Accessing and modifying date components (day, month, year)
   - Validating dates
   - Performing date calculations (e.g., adding/subtracting days, weeks, months, years)
   - Retrieving date-related information (e.g., day of the week, number of days in a month/year)
   - Printing month and year calendars

2. **Input Validation**: The `clsInputValidate` class provides utility functions for validating user input, such as:
   - Checking if a number is within a specified range
   - Checking if a date is within a specified range
   - Reading integer and floating-point numbers from the user
   - Generating random numbers, characters, and keys

3. **String Manipulation**: The `clsString` class offers a variety of string manipulation functions, such as:
   - Counting words, letters, and vowels in a string
   - Transforming the case of characters (uppercase, lowercase, title case)
   - Splitting and joining strings
   - Removing punctuation
   - Replacing words in a string

4. **Utility Functions**: The `clsUtility` class provides additional utility functions, such as:
   - Generating random numbers and characters
   - Swapping values of different data types
   - Encrypting and decrypting text
